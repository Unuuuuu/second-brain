# 해외 유저 구글 광고 노출 수정

## 배경

- 제휴 광고는 국내 사용자에게만 국한되는 경우가 대부분이므로, 해외 사용자에게는 미노출하고자함
    - 해외 사용자의 트래픽은 허수 트래픽에 가까워, 성과에 악영향을 미칠 가능성이 높음

## 현재 상황

- 해외 사용자는 디바이스의 regionCode ≠ KR로 분류함
- 해외 사용자는 임팩트 메인팝업(제휴 광고) 대신 Google Admob Opening Ads가 노출됨
- 해외 사용자에게 배너4종, 전면팝업4종 CPC 광고가 노출됨

## 액션 아이템

- [ ] 배너 광고 → 해외 사용자에게는 항상 네트워크 광고를 노출함
- [ ] 전면팝업 광고 → 해외 사용자에게는 항상 네트워크 광고 노출함
    - property: skippable 속성 부여해서, 5초 이내 광고만 보여주는것이 가능하다면 네트워크 광고 노출
    - 5초 이상의 긴 광고만 보인다면, 미노출 방향으로 수정해도됨

## 참고사항

- 1일자 진급축하팝업, 전역축하팝업은 광고상품에서 제거하고, 인하우스 홍보용으로만 사용 예정이어서, 기존대로 유지해도 무방함
