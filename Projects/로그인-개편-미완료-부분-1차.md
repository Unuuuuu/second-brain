# 로그인 개편 미완료 부분 1차

## 개요

로그인 개편 작업 중 미완료된 부분을 파악하고 재진님께 인수인계할 내용 정리.

## 미완료 작업 중 일부

- [ ] 로그인 문제 해결 화면에 대한 다국어 처리
- [ ] 계정 찾기 플로우 일부
    - [ ] 계정 찾기를 눌렀을 때 언어 종류에 따라서 Dialog 노출
    - [ ] 계정을 찾기 위해 본인 인증을 할 수 있다.
    - [ ] 본인 인증 후 계정 찾기 완료 결과 화면으로 이동한다. (API 연동 제외. API 아직 마련되지 않음.)
- [ ] 이메일 로그인 플로우
    - [ ] 새로운 이메일로 이메일 인증 플로우를 탄 다음에 로그인할 수 있다.

## 참고 사항

- 다음 세 화면에서 이메일 인증 기능이 필요합니다. 감안해서 작업이 필요합니다.
    - 이메일 로그인 중에 이메일 인증
    - 비밀번호 재설정 중에 이메일 인증
    - 로그인 수단 관리 이메일 변경 중에 이메일 인증
